# Trello - uso rápido

## 1) Criar cards em lote por lista
1. Abra a lista `Feito` no Trello.
2. Copie todo o conteúdo de `01-feito-titulos.txt`.
3. Cole dentro da lista no Trello (cada linha vira um card).
4. Repita na lista `A Fazer` com `02-a-fazer-titulos.txt`.

## 2) Preencher descrição dos cards
1. Abra `03-cards-detalhes.md`.
2. Para cada card, copie os campos e cole na descrição.

## 3) Organização por sprint (opcional)
- Use os arquivos `sprint-S1-titulos.txt` até `sprint-S7-titulos.txt` para criar listas/sprints.
